Подробное описание

Абстрактый класс, предназначенный для генерации выборки гипергеометрического распределения

Данный класс позволяет генерировать выборку для гипергеометрического распределения с заданными параметрами и размером. Класса имеет три наследника, реализующие метод обратной функции, метод испытаний Бернулли и табличный метод обратной функции. Также класс позволяетs получить полную информацию моделируемом распределении и методе моделирования.

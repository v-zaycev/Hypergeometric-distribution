Подробное описание

Наследник класса Hypergeom_sample, реализующий метод обратной функции.

Данный класс позволяет генерировать выборку для гипергеометрического распределения с заданными параметрами и размером методом обратной функции. Также класс позволяет получить полную информацию моделируемом распределении и методе моделирования.
